## 📖 Overview

This repository contains solutions for **Assignment 2** of the Python programming course.

The assignment focuses on understanding **Control Structures in Python**, including:

- Conditional Statements (`if-else`)
- Exception Handling (`try-except`)
- Iterative Statements (`for` loop)
- Arithmetic Operations
- User Input Handling

The programs are written in a simple and beginner-friendly manner.

---

## 📌 Assignment Tasks

### ✅ Task 1: Even or Odd Number Checker

### Objective
Create a program that:

- Accepts an integer from the user.
- Determines whether the number is even or odd.
- Handles invalid (non-numeric) inputs using exception handling.

### Concepts Covered

- User Input
- `if-else`
- Modulus Operator (`%`)
- `try-except`
- `ValueError`

### Sample Output

```
Enter a number: 12
Given Number is even 12
```

```
Enter a number: 15
Given Number is odd 15
```

```
Enter a number: hello
Invalid input! Please enter numeric values only.
```

---

### ✅ Task 2: Sum of Integers from 1 to 50

### Objective

Write a program that:

- Uses a `for` loop.
- Calculates the sum of integers from **1 to 50**.
- Displays the final result.

### Concepts Covered

- `for` Loop
- `range()`
- Variables
- Arithmetic Operations

### Sample Output

```
Sum of first 50 integers: 1275
```
